Welcome to the Big Build

The Big Build is a complete static site framework that uses Gulp, Webpack and Metalsmith.

Built for Website Performance

From the ground up, Big Build uses many of the best practices that lead to site optimization. HTML, CSS and JavaScript files are optimized by default. Additionally, Big Build supports lazy-loading, critical css, and inline imaging right out of the box.
